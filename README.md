network-monitor
===============

Monitor your network uptime and downtime.


How-to-use
---------------
Enter network-monitor folder in terminal and execute

```
chmod +x monitor.sh
./monitor.sh
``` 


Working
---------------
1. Logs are monitored into 'pinglog' file in the script folder.
2. Closing the terminal would stop monitoring
